# Mfumo wa Auto Update

App hii inatumia GitHub Releases kupitia electron-updater.

Repo iliyowekwa kwenye package.json:
- owner: issanass
- repo: nass-mazao-255

Ili kutoa update mpya:
1. Badili code au ongeza feature.
2. Pandisha version kwenye package.json, mfano 2.0.1.
3. Commit na push GitHub.
4. Tengeneza tag mpya, mfano v2.0.1.
5. GitHub Actions itajenga installer na kuiweka kwenye Releases.
6. App iliyopo kwenye PC itaona update na kuipakua.

Muhimu: Data ipo kwenye SQLite database ndani ya userData folder, si ndani ya app files. Hivyo update haitafuta data.
